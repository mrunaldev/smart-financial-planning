# Smart Financial Planning

A responsive dark-themed financial planning app with tabbed sections for budget, goals, expenses, investments, insurance, cards, net worth, tax planning, gifts, misc, and one-time budgets.

## Structure

- `index.html` - App markup
- `assets/css/styles.css` - Responsive styling
- `assets/js/app.js` - Local storage, tabs, totals, search, and entry actions

## Run Locally

```bash
python3 -m http.server 8082
```

Then open:

```text
http://localhost:8082
```
